# LF3DP
Hold mouse left button to draw <br>
Release mouse left button to finish or cancel <br>
A,D: switch paintor color <br>
W,S: switch background color <br>
R: reverse the last painting <br>
C: clear canvas <br>
H: hide painter <br>
